The Meat and Potatoes

(don't wanna drink on an empty stomach)

  1. Starting from scratch, open terminal, make a 'sassy-bartender' file: mkdir sassy-bartender
  2. In terminal: cd sassy-bartender Create the index page: touch index.html
  3. Create a styles folder: mkdir styles
  4. cd into styles, and create two files: touch bartend.css bartend.scss

Get Sassy.

Which brings me to Sass. Sass is an industry standard, a CSS pre-processor. Sass brings much of the functionality that we've all grown to love in more functional programming languages into CSS. Functionality like variables, mix-ins, inheritance.
What's not to like!

You utilize Sass by working in one of two types of files (.scss, or .sass), writing Sass's shorthand. The gem then translates the shorthand into complete CSS. In my limited time working with Sass, I've found I prefer working .scss files over .sass, as they're a bit closer syntactically to the .css we all love to um... love.
I'll walk you through the installation process on OS X to get it running on our sassy project. If you're running a different OS, visit the site, docs are excellent:

Install and Link

  1. Open up terminal. And type gem install sassNow feel the sass. Growl.
  2. cd into our sassy-bartender/styles, and set the Sass gem to work on our style sheets: sass --watch bartend.css:bartend.scss

Set a Watch

We just linked our two style sheets. Sass will be watching any changes we make to bartender.scss
The .scss format enables the mixins and shortcuts we like SO MUCH, the Sass gem translates this shorthand (which doesn't yet exist in CSS itself) to our bartender.css file. EASY PEASY.

I'll Take a Manhattan, Bartender

Installing cool gems and putting the files in the right places.

Now we start mixing drinks. We're going to download a set of design gems, that let us design basic web pages more elegantly, and (hopefully) easily. Let's download the 3 gems:

  1. Enter Terminal, and install these three gems. For gem installation, the directory where you run these commands is irrelevant:
    gem install bourbon
    gem install neat
    gem install bitters
  2. If you're not there already, cd into the project styles folder. Now install the files into your styles folder. This is not gem installation, rather this imports Thoughtbot's liquor themed templates, mixins and tools into our sassy-bartender:
    bourbon install
    neat install
    bitters install
  3. In your text editor, the HEAD of your index.html file, add these 2 style links in this order: <link rel="stylesheet" src="http://necolas.github.io/normalize.css/3.0.1/normalize.css">
    <link rel="stylesheet" href="styles/bartend.css">

normalize.css exists to wipe the design slate clean across any browser platform. We want to link here first in our header, before links to our active design with the Bourbon based tools.

Walk in a straight line sir...

Import Thoughtbot's tools in the correct order so they behave.

  1. Inside bartend.scss in your text editor, we'll make our first imports that will let us do awesome stuff faster: @charset "utf-8";
    @import 'bourbon/bourbon';
    @import 'base/grid-settings';
    @import 'neat/neat';
    @import 'base/base';
  2. Inside the styles directory, open the 'base' or 'bitters' folder (depending on how your system downloads and assigns the name), and open the file
    _grid-settings.scss
    In line 1, make the change noted if you are designing a static site rather than a rails app.
  3. And save...
